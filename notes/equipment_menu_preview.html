Equipment Menu Redesign

This is a click-through preview of the proposed sidebar redesign. Nothing in the live app (RoomCalculator.html) has changed yet.

Try it: click the pills, type in the search (try mic, table, bar), and narrow your window to see the horizontal scroll chevrons appear.

What changed

  • One inline search bar at the top, replacing the four floating round .btnItemSearch buttons.
  • Horizontally-scrolling category pills replacing the four hardcoded .subtablinks sub-tabs, with chevron buttons that appear only when the row overflows.
  • New "Custom Items" pill sourced from the VRC Custom Item Library (IndexedDB) — same data the Space-bar Quick Add modal uses.

Search behaviour

The search filters tiles across all categories in place — no modal. If the currently-active pill has no matches but another pill does, the active pill auto-switches to the best-matching panel.

Quick Add modal stays

Space and the canvas right-click menu still open the existing full Quick Add palette. Power users keep that entry point; only the four floating round search buttons go away.

How to test the real implementation

Once you approve this design, the production changes go into:

  • RoomCalculator.html — replace markup at lines 245–248 & 254/311/366/407; add Custom Items panel.
  • style.css — delete .btnItemSearch rules; add the new pill / search-bar rules.
  • js/roomcalc.js — refactor openSubTab(), add onEquipmentSearchChange(), populate Custom Items tiles from the library cache.
